{% if lang == 'en' %} Three ISO standards, certified by an accredited body.     {# --- ISO/IEC 27001:2022 --- #}

  1. {% if lang == 'en' %}Information Security Management{% else %}Sistema di Gestione della Sicurezza delle Informazioni{% endif %}

    ISO/IEC 27001:2022

    {% if lang == 'en' %} The international reference standard for an Information Security Management System (ISMS). Our ISMS covers all 93 controls of Annex A, with applicability declared in the Statement of Applicability and a risk treatment plan that ties each control to a specific threat scenario. {% else %} Lo standard internazionale di riferimento per un Sistema di Gestione della Sicurezza delle Informazioni (SGSI). Il nostro SGSI copre tutti i 93 controlli dell'Allegato A, con applicabilità dichiarata nello Statement of Applicability e un piano di trattamento del rischio che lega ciascun controllo a uno scenario di minaccia specifico. {% endif %}

    {% if lang == 'en' %}Scope{% else %}Scope{% endif %}
    {% if lang == 'en' %}Advisory services, secure software development, SaaS operation, customer-data handling.{% else %}Servizi di advisory, sviluppo software sicuro, operatività SaaS, trattamento dei dati cliente.{% endif %}
    {% if lang == 'en' %}Version{% else %}Versione{% endif %}
    2022 — {% if lang == 'en' %}latest revision, replaces 27001:2013/2017{% else %}revisione corrente, sostituisce 27001:2013/2017{% endif %}
    {% if lang == 'en' %}Controls{% else %}Controlli{% endif %}
    {% if lang == 'en' %}93 across four themes — organizational, people, physical, technological{% else %}93 articolati in quattro temi — organizzativi, persone, fisici, tecnologici{% endif %}
    2. {# --- ISO/IEC 27017:2015 --- #}

  2. {% if lang == 'en' %}Cloud-Specific Security Controls{% else %}Controlli di Sicurezza Cloud{% endif %}

    ISO/IEC 27017:2015

    {% if lang == 'en' %} The cloud-security companion to ISO/IEC 27001 — it extends the ISMS with seven cloud-specific controls and additional guidance for the 27002 set. It explicitly distinguishes the obligations of the Cloud Service Customer (CSC) from those of the Cloud Service Provider (CSP). We operate as both, and we hold each role to the standard. {% else %} Il companion di ISO/IEC 27001 per la sicurezza cloud — estende l'ISMS con sette controlli cloud-specifici e linee guida aggiuntive per il set 27002. Distingue esplicitamente gli obblighi del Cloud Service Customer (CSC) da quelli del Cloud Service Provider (CSP). Noi operiamo in entrambi i ruoli, e ciascuno è tenuto allo standard. {% endif %}

    {% if lang == 'en' %}Our role{% else %}Ruolo{% endif %}
    {% if lang == 'en' %}CSC + CSP — we consume cloud services and we deliver them{% else %}CSC + CSP — consumiamo servizi cloud e li eroghiamo{% endif %}
    {% if lang == 'en' %}Coverage{% else %}Copertura{% endif %}
    {% if lang == 'en' %}Applied on top of ISO/IEC 27001:2022 via the cloud-specific section of the Statement of Applicability{% else %}Applicato sopra ISO/IEC 27001:2022 tramite la sezione cloud-specifica dello Statement of Applicability{% endif %}
    {% if lang == 'en' %}Topics{% else %}Aree{% endif %}
    {% if lang == 'en' %}Shared responsibility, tenant isolation, virtual environment hardening, administrative ops, cloud monitoring, alignment of cloud agreements with security policy{% else %}Responsabilità condivisa, isolamento dei tenant, hardening dell'ambiente virtuale, operatività amministrativa, monitoraggio cloud, allineamento dei contratti cloud alla policy di sicurezza{% endif %}
    3. {# --- ISO 9001:2015 --- #}

  3. {% if lang == 'en' %}Quality Management{% else %}Sistema di Gestione della Qualità{% endif %}

    ISO 9001:2015

    {% if lang == 'en' %} The reference standard for quality management. The current version is amended by Amendment 1:2024 (climate-action changes), which is reflected in our Quality Management System. The QMS is integrated with the ISMS into a single Integrated Management System — one set of objectives, one set of audits, one set of evidence. {% else %} Lo standard di riferimento per la gestione della qualità. La versione corrente è integrata dall'Amendment 1:2024 (climate-action changes), che è rispecchiato nel nostro Sistema di Gestione della Qualità. Il SGQ è integrato con il SGSI in un unico Sistema di Gestione Integrato — un solo set di obiettivi, di audit, di evidenze. {% endif %}

    {% if lang == 'en' %}Scope{% else %}Scope{% endif %}
    {% if lang == 'en' %}Repeatable, measurable, continuously improving delivery of advisory services and software products{% else %}Erogazione ripetibile, misurabile e in miglioramento continuo dei servizi di advisory e dei prodotti software{% endif %}
    {% if lang == 'en' %}Version{% else %}Versione{% endif %}
    2015 + Amd.1:2024 ({% if lang == 'en' %}climate action{% else %}azione climatica{% endif %})
    {% if lang == 'en' %}Integration{% else %}Integrazione{% endif %}
    {% if lang == 'en' %}Operated jointly with ISO/IEC 27001:2022 and ISO/IEC 27017:2015 in a single Integrated Management System{% else %}Esercito congiuntamente a ISO/IEC 27001:2022 e ISO/IEC 27017:2015 in un unico Sistema di Gestione Integrato{% endif %}
